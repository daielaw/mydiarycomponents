Untuk mengaplikasikan tampilan window yang adaptive bisa pakai "WindowSizeClass" dengan menambahkan dependency

commonMain.dependencies {
    implementation("org.jetbrains.compose.material3.adaptive:adaptive:1.2.0-alpha06")
}

ref: https://www.jetbrains.com/help/kotlin-multiplatform-dev/compose-adaptive-layouts.html#using-window-size-classes
