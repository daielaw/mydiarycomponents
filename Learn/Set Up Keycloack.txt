1. Buat Realm baru : LaksanaNeoRealm

2. Buat Client baru : LaksanaNeoWebAPI , setting Grant access saja dulu

3. Buat Client Scope baru : LaksanaNeoWebAPIScope, centang include token

4. Menambah mapper di token
 -mapper audience
Masuk Client scopes -> ke namascope (LaksanaNeoWebAPIScope) -> add mapper by configuration -> pilih Audience-> beri nama : LaksanaNeoWebAPIAudience

- mapper roles
Masuk ke client scopes -> ke roles (di halaman lain jika gk ada di list) -> berikan nama roles

* Sebelum menampahkan token ini tambahkan dahulu di Realm Setting -> User profile -> create attribute -> berikan nama dan permission untuk disesuaikan

 - mapper nik, departement, mobile phone
Masuk Client scopes -> ke namascope (LaksanaNeoWebAPIScope) -> add mapper by configuration-> pilih user attribute -> tambahkan nik, departement, dan mobile phone

5. Buat Roles Realm dengan nama Administrator
Realm roles -> create role

6. Tambahkan Roles Realm lagi misal LKSUser
Realm roles -> create role


9. Buat user baru 
Users -> add user,
dan assign role tiap user Administrator /LKSUser

10. pengecekan
- POST
http://localhost:8080/realms/LaksanaNeoRealm/protocol/openid-connect/token
